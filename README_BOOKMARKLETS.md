# PJEPLUS BOOKMARKLETS

## 📦 Arquivo Consolidado: `bookmarklets.js`

Este arquivo contém todas as funcionalidades bookmarklet organizadas em um único local, com interface unificada e fácil acesso.

### 🚀 Funcionalidades Disponíveis

1. **📝 MinutaPJE** - Automação completa de minutas no PJE
2. **📋 SISBAJUD Ordens** - Extrator de ordens judiciais com bloqueio positivo
3. **📊 Séries Teimosinha** - Extrator de séries elegíveis do SISBAJUD

### 📥 Como Usar

1. **Download**: Baixe o arquivo `bookmarklets.js`
2. **Instalação**: Arraste o arquivo para a barra de favoritos do seu navegador
3. **Execução**: Clique no bookmarklet quando estiver na página apropriada

### 🎯 Interface Unificada

O bookmarklet cria um botão flutuante "🚀 PJE+" no canto inferior esquerdo da página. Clique nele para abrir um menu com todas as funcionalidades disponíveis.

### 📁 Organização

- **`bookmarklets.js`** - Arquivo principal consolidado (38KB)
- **`temper/`** - Pasta com versões Tampermonkey organizadas
  - `temper-minutapje.user.js` - Versão Tampermonkey do MinutaPJE
  - `temper-sisbajud-ordens.user.js` - Versão Tampermonkey do extrator SISBAJUD
