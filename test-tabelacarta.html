🧪 Teste do Assistente de Documentos PJe

📋 Como testar:

  1. Execute o bookmarklet (cole o código na console ou carregue como bookmark)
  2. Selecione um ID de documento abaixo
  3. Observe o indicador visual que aparece
  4. Clique no indicador ou pressione Ctrl+D para abrir o menu
  5. Teste os botões do menu contextual

💡 Dica: Os botões "Copiar eCarta" e "📄 Docs" devem aparecer no canto inferior direito.

🎯 IDs de Documento para Teste

Selecione qualquer um dos IDs abaixo:

  • ID simples: 1234567890123
  • Com prefixo: doc: 9876543210987
  • Documento completo: documento: 1122334455667
  • ID com separador: ID: 5544332211009
  • UUID: A1B2C3D4-E5F6-7890-ABCD-EF1234567890

Texto com ID integrado: "O documento número 1357924680135 foi anexado ao processo."

🔍 Cenários de Teste Avançados

IDs em diferentes contextos:

  • Em uma frase: "Conforme documento 9988776655443, o processo foi arquivado."
  • Lista numerada: 1. Documento 1111222233334 - Petição inicial
  • Entre parênteses: (doc: 5555666677778)
  • Com hífen: documento-4444555566667

IDs que NÃO devem ser detectados (muito curtos):

  • ID curto: 123456789 (9 dígitos)
  • Número comum: 12345
  • Telefone: (11) 99999-9999

⚙️ Funcionalidades Esperadas

✅ Deve funcionar:

  • Detectar IDs de 10-15 dígitos quando selecionados
  • Mostrar indicador visual temporário
  • Abrir menu contextual ao clicar no indicador
  • Responder ao atalho Ctrl+D
  • Copiar ID para clipboard
  • Tentar abrir documento no PJe
  • Mostrar botão de ajuda com instruções

🚫 Não deve detectar:

  • IDs muito curtos (menos de 10 dígitos)
  • IDs muito longos (mais de 15 dígitos)
  • Texto que não contenha padrões válidos
