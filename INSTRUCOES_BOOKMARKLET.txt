2. COMO CRIAR O BOOKMARKLET:

No Chrome/Edge/Firefox:
1. Pressione Ctrl+Shift+O (ou vá em Favoritos > Gerenciar favoritos)
2. Clique em "Adicionar novo favorito" ou "Nova pasta" > "Novo favorito"
3. Nome: "Análise eCarta" 
4. URL: COLE O CÓDIGO ACIMA (toda a linha começando com "javascript:")
5. Salve

3. COMO USAR:

1. Acesse uma página de processo no PJe
2. Clique no bookmarklet "Análise eCarta" na barra de favoritos
3. Escolha a data (2 dígitos do dia) ou deixe vazio para data mais recente
4. Aguarde o processamento automático
5. O resultado aparecerá em uma tabela que pode ser copiada

4. FUNCIONALIDADES:

✓ Seleção de data específica ou mais recente
✓ Busca automática de intimações de correio na timeline
✓ Login automático no eCarta
✓ Correlação de dados entre PJe e eCarta
✓ Tabela HTML com resultados
✓ Botão para copiar tabela
✓ Mensagens de status e erro
✓ Logs de debug no console (F12)

5. REQUISITOS:

- Estar logado no PJe
- Página de detalhes do processo aberta
- Internet ativa para acessar eCarta
- Credenciais eCarta configuradas no código
