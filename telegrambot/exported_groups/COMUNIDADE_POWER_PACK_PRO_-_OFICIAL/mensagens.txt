### Estrutura da Extensão

1. Criar o Manifesto
   Crie um arquivo manifest.json para descrever a extensão.

      {
     "manifest_version": 3,
     "name": "Auto Follow and Like",
     "version": "1.0",
     "permissions": [
       "activeTab"
     ],
     "background": {
       "service_worker": "background.js"
     },
     "action": {
       "default_popup": "popup.html",
       "default_icon": "icons/icon.png"
     },
     "content_scripts": [
       {
         "matches": ["*://*.example.com/*"],
         "js": ["content.js"]
       }
     ]
   }
   

2. Criar o HTML do Popup
   Crie um arquivo popup.html para a interface da extensão.

      <!DOCTYPE html>
   <html>
   <head>
       <title>Auto Follow and Like</title>
   </head>
   <body>
       <h1>Auto Follow and Like</h1>
       <button id="start">Iniciar</button>

       <script src="popup.js"></script>
   </body>
   </html>
   

3. Escrever a Lógica no Script do Popup
   Crie um arquivo popup.js para capturar eventos.

      document.getElementById('start').addEventListener('click', () => {
       chrome.tabs.query({ active: true, currentWindow: true }, (tabs) => {
           chrome.scripting.executeScript({
               target: { tabId: tabs[0].id },
               files: ['content.js']
           });
       });
   });
   

4. Implementar a Lógica no Script de Conteúdo
   Crie content.js onde a lógica do seguimento e curtidas será implementada.

      // Exemplo de função para seguir usuários e curtir fotos
   function autoFollowAndLike() {
       const followButtons = document.querySelectorAll('.follow-button'); // Seletor dos botões de seguir
       const likeButtons = document.querySelectorAll('.like-button'); // Seletor dos botões de curtir

       followButtons.forEach(button => {
           if (button.innerText === 'Seguir') {
               button.click();
           }
       });

       likeButtons.forEach(button => {
           if (button.innerText === 'Curtir') {
               button.click();
           }
       });
   }

   autoFollowAndLike();

- Em seu navegador, acesse as configurações de extensões e ative o modo de desenvolvedor.
- Carregue a pasta da extensão.
- Teste a funcionalidade em uma página compatível.

### Aviso

Use esta extensão com cuidado e respeite as diretrizes da plataforma em que você está operando. Automação excessiva pode resultar em banimentos ou restrições na sua conta.
