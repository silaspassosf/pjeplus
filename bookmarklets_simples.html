📋 Bookmarklets Simples - Atos Judiciais

Automatiza apenas PEC, Prazo e Movimento para cada ato judicial

🔧 Como Usar:

  1. Use os bookmarklets prontos abaixo OU crie um personalizado no formulário
  2. Copie o código clicando no botão "Copiar"
  3. No seu navegador, crie um novo favorito/marcador
  4. Cole o código como URL do favorito
  5. Na tela de minuta do PJe, APÓS inserir o modelo, clique no bookmarklet

⚠️ Importante: Use apenas APÓS inserir o modelo do ato na tela de minuta do PJe!
