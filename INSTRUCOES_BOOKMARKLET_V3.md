# 📧 BOOKMARKLET CARTA FINAL V3.0 - INSTRUÇÕES COMPLETAS

## 🚀 Como instalar:

1. **Copie todo o código** da seção "CÓDIGO DO BOOKMARKLET" abaixo
2. **Crie um novo favorito** no navegador (Ctrl+D)
3. **Edite o favorito** e **substitua a URL** pelo código copiado
4. **Nomeie** como "📧 Carta eCarta" 
5. **Salve** na barra de favoritos

## 📋 Como usar:

### 🔍 PASSO 1: No PJe
1. Abra a **página de detalhes do processo** no PJe
2. Clique na aba **"Linha do Tempo"**
3. **Clique no bookmarklet** (favorito criado)
4. Digite o dia (ex: 15) ou deixe vazio para usar o mais recente
5. Aguarde - o eCarta abrirá automaticamente

### 📧 PASSO 2: No eCarta
1. **Clique no mesmo bookmarklet** na aba do eCarta
2. Aguarde o login automático
3. Aguarde a análise completa (alguns minutos)
4. Veja o relatório detalhado

## 🐛 PROBLEMAS COMUNS:

**❌ "Timeline não encontrada"**  
→ Certifique-se de estar na aba "Linha do Tempo" do processo

**❌ "Campos de login não encontrados"**  
→ Aguarde a página do eCarta carregar completamente

**❌ Timeout nas consultas**  
→ Conexão lenta, tente recarregar e executar novamente

**❌ "Nenhum dado encontrado"**  
→ Execute primeiro no PJe, depois no eCarta

## ✅ DICAS:

- **Teste primeiro** em processo com poucos documentos
- **Aguarde** as mensagens do script
- **Não feche** as abas durante execução
- **Console F12** mostra logs detalhados para debug
