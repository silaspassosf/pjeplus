# Code Simplifier (DeepSeek)

Aplique os princípios de simplificação (preservar comportamento, clareza > concisão). 
- Edite apenas a função/método indicado.
- Após editar, execute `py -m py_compile arquivo`.
- Output máximo: 500 tokens.
